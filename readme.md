# Coding Challenge

## 1. Fizz Buzz

- Input: an integer
- Output: 
    - every integer from zero to input integer
        - positive integer?
        - upper limit?
    - display "fizz" for numbers divisible by 3
        - string length limit?
    - display "buzz" for numbers divisible by 5
    - display "fizzbuzz" for numbers divisible by 3 and 5
- Enhancements:
    - Input for divisors
        - upper limit? 
    - Input for displayed words
